## 📊 Проект: анализ работы приложения такси

> **Задача:**  
> Проанализировать сезонность заказов, пунктуальность водителей, распределение оценок и влияние платформы на успешность поездки.

### 🔧 Что сделано

- Проведена **очистка и объединение** таблиц `passengers` и `drivers` в Python (pandas). Построены графики в seaborn, выделены слабые места в метриках. Сформулированы гипотезы.
- Построен **дашборд в DataLens** с фильтрами по платформам, дням недели и статусам заказа.

## 📊 Проект: анализ работы приложения такси
> **Задача:**  
> Проанализировать поведение пользователей игрового сервиса: оценить успешность операций, влияние премиум-статуса и возраста, выявить проблемные платформы и причины низкого удержания.
## 🔧 Что сделано

- Проведена **очистка и объединение** таблиц `user_data` и `logs` (pandas). Проверены пропуски, дубликаты, типы данных.
- Рассчитаны ключевые метрики: **success rate**, активность пользователей, доля премиум, распределение по платформам.
- Выполнена **сегментация по возрастным группам** (18–24, 25–34, 35–49, 50+) и платформам.
- Проанализированы **интервалы между операциями** (среднее, медиана, визуализация выбросов).
- Проведён **анализ удержания (retention)** через распределение числа операций на пользователя.
- Построены графики: гистограммы, stacked bar charts, тепловая карта успеха по часам/платформам, boxplot интервалов.
- Выполнены **статистические тесты** (t-test) для проверки значимости различий между возрастными группами.

---

## 📈 Результаты

| Вывод | Цифры |
|-------|-------|
| **Общий success rate** | **73,1%** (3289 успешных операций из 4500) |
| **Критически низкое удержание** | **97,6%** пользователей совершают только 1 операцию и уходят |
| **Самая успешная возрастная группа** | **18–24 года: 77,9%** успеха |
| **Самая проблемная возрастная группа** | **35–49 лет: 72,1%** успеха (разница статистически значима, p<0.05) |
| **Премиум чаще покупают 50+** | **13,3%** премиум в группе 50+ против **11,0%** в группе 25–34 |
| **Лучшая платформа по успеху** | **Планшет – 74,4%**, но доля операций всего 5% |
| **Основная платформа** | **Телефон – 61%** операций, success rate 73,3% |
| **Среднее время между операциями** | **100 минут** (сильно скошено выбросами) |
| **Медианное время между операциями** | **28 минут** (типичный интервал) |
